# 세계관 추가 설정 (A안)

이 문서는 `fantasy-sim-core/src/main/resources/lore/world_bible_A.json`을 사람이 읽기 쉽게 요약한 버전입니다.  
(코드는 해당 JSON을 로드해서 **더미 AI 출력**의 장소/세력/신앙 표현과 분위기 문장을 일관되게 만듭니다.)

## 1) 고유명사(A안) 요약
- 대륙: **에르노스 대륙**
- 국가/권역
  - 중앙: **로데르 제국**
  - 서부: **사하리드 사막왕국**
  - 남부(7개 연합): **연해칠국 동맹**
  - 남부(강대 왕국): **카이렌 남부왕국**
  - 동부: **실바나르 엘프왕국** (폐쇄, 관문 통제)
  - 북부: **무주지** (설원 부족 + 드워프 광산길드)
- 기관/집단
  - 교단: **태양성 교단**
  - 악마숭배자: **월흑교**
  - 드워프: **산맥 광산길드 연합**

## 2) 세력 관계(핵심)
- 제국↔사막왕국: 이웃, 국지도발 수준의 긴장 유지
- 제국↔남부 연합: 친제국파 지원/정치모략, 연합 내부 당쟁
- 제국↔남부 강대 왕국: 무력으로 밀 수는 있으나 부담 → 완충지대(남부 연합) 유지
- 사막왕국↔남부 연합: 제국 개입 시 반제국파 지원(역모략)

## 3) 신앙/마계 구조
- **주신**: 태양의 주신 **솔라**
- **4대 신**: 비타리아(생명) / 오필리아(풍요) / 아르콘(질서) / 리오나(흐름)
- **마왕**: 월왕 **루나르크**
- **4 고위마족(고위악마)**: 네크라(종언) / 카르니아(고갈) / 모르피오(왜곡) / 스태그논(정체)
- **4대 신 ↔ 4 고위마족**: 서로 반대 성질(대립)이며 상호 증오
- **상급악마(7대 죄악)**: 루미엘/바르곤/세릴/이르사/그라움/라카스/누몰  
  → “상급악마(7죄악) → 중급악마 → 하급악마/마물” 위계

## 4) 운영 원칙(스토리 개연성)
- 민간인의 엘프 왕국 입국은 사실상 불가. 허가된 인간/심사 통과 상인만 관문 통과.
- 태양성 교단은 정치적으로 중립(포교 목적). 목표는 마왕 제거.
- 마경(마계) 크기는 유지되며, 하급악마/마물이 새어나와 **귀족 유혹(힘/부)** → 귀족+월흑교 협력.
- 마물은 지능이 낮아 물질 피해 중심, 하급악마는 지능이 있어 모략 포함.
- 북부 수인: 반악마파(인간과 교류) / 친악마파(힘 추구, 인간 악마숭배자도 불신)

## 5) 태그(게임/AI 계약)
- 태그는 `key:value` 형태의 **세계관 제약 장치**입니다. (예: `region:north`, `nation:empire`)
- 엔진이 먼저 태그를 뽑고(=세계관/정세에 맞는 사건 맥락 확정), AI/더미 AI는 **그 태그를 벗어나는 새 국가·새 신격·새 위계**를 만들 수 없습니다.
- 룰(effects.json)은 **turnType + tags + choiceId(+조건)** 조합으로 수치 변화를 계산합니다.

## 6) 추가 메모(원문 반영)
- 제국↔사막왕국 교류: 사막은 보석/비단/향신료(사치품), 제국은 밀(식량)과 제국제 공산품·도구류.
- 교단은 목표가 ‘마왕 제거’지만, **대륙 북부 설원으로 성기사를 대규모 파견하지는 않음**(활동권 내 정리 위주).
- 악마숭배자는 주로 북부 설원에 존재(인간 중심). 친악마파 수인과도 ‘같은 편’이라기보다 서로 껄끄러운 협력 관계.
- AI가 자유롭게 추가하더라도, 갑작스런 장르 붕괴(스팀펑크/차원침공/현대군 난입)는 금지.
